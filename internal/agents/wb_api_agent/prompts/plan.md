Build a machine-executable Wildberries ApiExecutionPlan.

Use only operations retrieved from the WB OpenAPI registry.

For every step:
- operation_id must exactly match the registry operation_id
- method must exactly match the registry method
- server_url must exactly match the registry server_url
- path_template must exactly match the registry path_template
- readonly must match x_readonly_method
- Authorization must use executor_secret with secret_name WB_AUTHORIZATION

Important: operation_id is the registry operation id. It may be generated by the registry loader if the source OpenAPI method has no operationId.
